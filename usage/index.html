Usage Guide

SceneShift is keyboard-centric. You can navigate the interface entirely without a mouse.

Navigation & Controls

Action Input Description
Navigate ↑ ↓ or k j Scroll through the app list.
Toggle Space Select/Deselect the highlighted app.
Select All a Select all visible apps.
Deselect All x Clear all current selections.
Find/Search Ctrl + f Search running apps to add them quickly.
Help ? Toggle the help menu overlay.

Modes & Actions

Action Input Description
Kill Mode K Terminate selected apps. Displays "RAM Reclaimed" stats.
Restore Mode R Relaunch apps (requires exec_path to be set).
Apply Preset 1 - 9 Instantly apply a saved preset (e.g., Gaming, Rendering).

Management & Editors

Action Input Description
New App n Add a new app to the tracking list.
Edit App e Edit the currently selected app configuration.
Delete App d Remove the selected app.
Theme Selector t Switch or edit themes live.
Preset Editor p Open the editor to add or modify presets.
