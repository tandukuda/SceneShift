# Troubleshooting

## Scene doesn't work

- Ensure SceneShift is running as Administrator
- Check process names carefully

---

## App closes unexpectedly

- You may have included it in a kill scene
- Review your preset configuration

---

## Nothing happens

- Some system processes cannot be controlled
- This is expected behavior

---

## Still stuck?

Open a GitHub issue with:
- SceneShift version
- Scene used
- Expected behavior
