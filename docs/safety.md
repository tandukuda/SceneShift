# Safety & Disclaimer

!!! warning "System Stability Warning"
    **SceneShift interacts directly with system processes.**
    
    While the application is built with safety in mind, terminating essential system applications or drivers may cause system instability, crashes, or data loss.
    
    - Always save your work in other applications before using Kill Mode.
    - Avoid terminating processes if you are unsure of what they do.
    - Use the "Suspend" feature if you are hesitant about killing a process entirely.
