Getting Started

SceneShift runs as a portable Windows executable.

Requirements

  • Windows 10 or newer
  • Administrator privileges (required for process control)

Installation

  1. Download the latest release from GitHub
  2. Extract the archive
  3. Run SceneShift.exe as Administrator

No installer. No background services.

First Launch

On first launch: - SceneShift creates its config directory - Default scenes are generated - You can immediately switch scenes

SceneShift does nothing automatically. All actions are explicit.

Safety Note

Kill mode terminates applications instantly. Save your work before switching scenes.
